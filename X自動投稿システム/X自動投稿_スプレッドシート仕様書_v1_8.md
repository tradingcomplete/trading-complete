### シート⑥ 学びログ

**役割**: 過去のトレード学びを蓄積し、投稿プロンプトの引き出しとして注入
**読み込み関数**: `getLearningLog_(postType, maxItems)`（geminiApi.gs）★v1.1でスコアベース選択に変更
**書き込み関数**: `saveLearningToLog_(category, learning, postType)`（geminiApi.gs）
**拡張関数**: `getLearningLogWithScores()`, `incrementLearningUsage()`（sheetsManager.gs）★v1.1追加
**注入先**: 10タイプ（件数別、下記参照）

```
列  ヘッダー      内容                          データ型
────────────────────────────────────────────────────────
A   日付          学びが記録された日付           yyyy/MM/dd
B   カテゴリ      6カテゴリのいずれか            テキスト
C   学びの内容    具体的な学び・気づき           テキスト（長文）
D   ソース        抽出元の投稿タイプ             テキスト
E   使用回数      プロンプトに注入された回数      数値（初期値0）  ★v1.1追加
F   有効度スコア   この学びが入った投稿の品質      数値（初期値0）  ★v1.1追加
G   最終使用日    最後に注入された日              yyyy/MM/dd      ★v1.1追加
H   ステータス    active/retired                 テキスト         ★v1.1追加
```

**選択ロジック（★v1.1で変更）**:
```
v1.0: 日付の新しい順にN件選択（ランダム性なし）
v1.1: スコアベース選択
  → 70%: 有効度スコア上位から選択（実績重視）
  → 30%: ランダムに選択（新しい学びにもチャンスを与える）
  → 同スコアなら使用回数が少ないものを優先
  → 選択された学びの「使用回数」を自動インクリメント
  → retired ステータスの学びはスキップ
```

**蓄積方法（2系統）**:
```
1. 自動蓄積: 投稿成功時にGeminiが投稿テキストから学びを1行抽出
   → extractPostInsights_() → saveLearningToLog_()
   → RULE_1〜4、KNOWLEDGEはスキップ（教訓から教訓の循環を防止）
2. 手動追記: コンパナがシートに直接書き込み
```

**6カテゴリ（自動判定）**:
```
相場心理 / テクニカル / ファンダメンタルズ / リスク管理 / 指標分析 / 市場構造
```

**タイプ別注入件数**:
```
投稿タイプ                     最大件数   理由
─────────────────────────────────────────────────
MORNING                        2         朝の見通しに直近の学びを反映
GOLDEN                         2         一日の振り返りに活用
WEEKLY_REVIEW                  5         週間振り返りに厚みを出す
WEEKLY_LEARNING                5         蓄積を活かした深い振り返り
WEEKLY_HYPOTHESIS              3         過去の仮説検証を次の仮説に
RULE_1〜4                      各3       カテゴリの合う学びを引き出しに
TOKYO,LUNCH,LONDON,NY,INDICATOR なし     速報系は不要
NEXT_WEEK                      なし      イベント整理のため不要
KNOWLEDGE                      なし      知識解説のため不要
```

**安全設計**:
```
・重複チェック: 直近20件と完全一致する場合はスキップ
・エラー耐性: 学びログ抽出のエラーで投稿フロー全体は止まらない
・ソート: 日付の新しい順にソートして最新のものから注入
```

**スコープ別注入ルール**:
```

───────────────────────────────────────────────────
```

**データ更新方法（2系統）**:
```
方法1（推奨）: 外為どっとコム → コピペ → インポート ★v1.4で詳細確定
  ① 「経済指標_貼り付け」シートのA3セルに外為どっとコムのデータを貼り付け
  ② importFromRawSheet() を実行（メニュー「経済指標をインポート」または直接実行）
  ③ 経済カレンダーに自動変換・書き込み（インポート後に貼り付けシートは自動クリア）
  ④ 重要な指標（CPI・GDP・雇用統計等）は手動でG列を「高」に変更

方法2: 手動でシートに直接入力
```

**貼り付けデータの行構造（外為どっとコム形式）**:
```

```
