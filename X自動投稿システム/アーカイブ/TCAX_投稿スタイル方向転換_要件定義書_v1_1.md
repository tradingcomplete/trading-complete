## 2. やってみて分かったこと（v1.0ドラフトにはなかった学び）

### 学び1: ストーリー主導とデータ忠実性の両立

```
v1.0の懸念: 「ストーリー主導にするとGeminiがハルシネーションしやすくなる」
v12.4の結果: Claudeに変更したことで解決。Claudeはデータに忠実にストーリーを書ける。
教訓: ストーリー主導の問題ではなく、Geminiの特性が問題だった。
```

### 学び2: クロスチェックより確定データ

```
v1.0の前提: 「4層安全網は変更しない」「クロスモデル検証が品質を担保」
v12.4の結果: クロスチェックは検出はできても修正ができない場合がある。
  さらに、修正ステップが正しい投稿を壊すケースが発生。
  確定データ（通貨強弱）をアンカーとする方が確実。
教訓: 異モデル検証 < 正しいデータの保持。引き算も大事。
```

### 学び3: 体言止めでキャラクターが際立つ

```
v1.0には未記載: 絵文字行のスタイルについて触れていなかった
v12.4の結果: 体言止め（速報調）+ →行（コンパナ調）の二層構造で、
  メリハリのある投稿が生まれるようになった。
  「📝米イラン交渉が決裂。」←速報 →「ホルムズ海峡をめぐる緊張が再燃ですね。」←人間味
教訓: 速報的な事実提示と人間味のある解説を構造的に分離する。
```

### 学び4: LLMは最末尾の指示を最も重視する

```
v1.0には未記載: キャラクターシートの配置問題
v12.4の結果: キャラクターシートはプロンプト冒頭にあるが、
  24,000文字のデータを処理するうちに口調が薄まる。
  最末尾にキャラクターリマインダーを配置して解決。
教訓: 重要な指示はプロンプトの冒頭AND最末尾に配置する。
```

## 3. Before / After（実際の投稿比較）

### Before（v12.3 Gemini生成）
```
📝豪ドルが強い一方、米ドルが軟調。
→豪州の政策金利が高水準で推移していることで、金利差が意識されている感じかなと。
```
問題: データ主導。「だから何？」が伝わらない。

### After（v12.4 Claude生成）
```
☕米イラン協議が決裂、ホルムズ海峡封鎖が長期化し原油が100ドル超え。
→豪ドル米ドルは上昇基調。
豪ドル米ドル 0.7093まで到達。

📝日銀、中東情勢を注視する姿勢を示す。
→4月利上げ観測が後退し、円の金利メリットは薄いまま。
AUDは前日比+2.80%と突出した強さで、ドル安も重なって0.7095ドル台まで上昇しています。
```
改善: ストーリー主導。因果チェーン。体言止め。方向が正確。
