## 1. 概要

### 1.1 現状の問題

エントリー時の感情・心理状態が **自由記入式（textarea）** のため:

- 表記揺れが発生（「冷静」「落ち着いていた」「calm」等が混在）
- 集計・分析が不可能（StatisticsModule/ReportModule/AISummaryModule のどれも未活用）
- AI分析に渡しても構造化されていないため精度が低い
- 未入力率が高い（自由記入は心理的ハードルが高い）

### 1.2 改善後

**単一選択（タップ/クリックで1つ選ぶ）+ 自由メモ** に変更:

- 単一選択 → 「今、一番強い感情は？」にワンタップで答える
- 1トレード=1感情で集計がクリーン（重複なし）
- 自由メモ（任意）→ 補足したい場合のみ
- AI分析 → 「冷静時の勝率73%、焦り時の勝率20%」が正確に自動算出
- 感情選択は **必須**（未選択ではエントリー不可 → 分析データの一貫性を保証）

### 1.3 なぜ単一選択か

| 方式 | メリット | デメリット |
|------|---------|-----------|
| 複数選択 | 「冷静だけど少し焦りも」を表現可能 | 集計に重複が発生、選択に迷う、データ量が少ないと統計的に意味が出にくい |
| **単一選択** | **集計がクリーン、選択が早い、AIの分析が明快** | 複合感情を表現しにくい |

**結論**: エントリー前のスピードが最優先。迷う時間は不要。「一番強い感情」を選ぶだけ。補足は自由メモで。

## 2. 選択肢の設計

### 2.1 感情カテゴリ（8種類）

| キー | 日本語ラベル | 絵文字 | 分類 | 説明 |
|------|------------|--------|------|------|
| `calm` | 冷静 | 😌 | ポジティブ | 落ち着いて判断できている |
| `confident` | 自信あり | 💪 | ポジティブ | エントリー根拠に自信がある |
| `focused` | 集中 | 🎯 | ポジティブ | チャート分析に集中できている |
| `cautious` | 慎重 | 🛡️ | ポジティブ | リスクを意識して判断している |
| `rushed` | 焦り | ⚡ | ネガティブ | 早くポジションを取りたい |
| `anxious` | 不安 | 😟 | ネガティブ | 不安要素がある中でのエントリー |
| `excited` | 興奮 | 🔥 | ネガティブ | テンションが上がりすぎている |
| `uncertain` | 迷い | 🤔 | ネガティブ | エントリーに迷いがある |

### 2.2 選択肢の根拠

- **8種類**: モバイルで2行×4列に収まる。多すぎず少なすぎない
- **ポジティブ4 + ネガティブ4**: AI分析で「ポジティブ/ネガティブ別勝率」が1対1対応
- **単一選択**: 「一番強い感情」に絞ることでデータの一貫性を確保
- **必須選択**: バリデーション対象（感情なしではエントリー不可。分析データの品質を保証）

## 3. データ構造

### 3.1 保存形式

**localStorage（trade オブジェクト内）:**

```javascript
// 新形式
entryEmotion: {
    selection: "calm",       // 選択されたキー（1つ、または空文字）
    memo: "特に問題なし"      // 自由メモ（任意、空文字可）
}

// 旧形式（既存データ）
entryEmotion: "冷静に分析できていた"  // プレーンテキスト
```

**Supabase（entry_emotion カラム）:**

現在の `TEXT` 型はそのまま維持。JSON文字列として保存する。

```sql
-- 新形式（JSON文字列）
entry_emotion = '{"selection":"calm","memo":"特に問題なし"}'

-- 旧形式（プレーンテキスト）※既存データ
entry_emotion = '冷静に分析できていた'
```

### 3.2 後方互換性

旧データを壊さないため、読み取り時に形式を自動判定する:

```javascript
function normalizeEmotion(raw) {
    if (!raw) return { selection: '', memo: '' };
    
    // 新形式（オブジェクト）
    if (typeof raw === 'object' && raw.selection !== undefined) {
        return { selection: raw.selection || '', memo: raw.memo || '' };
    }
    
    // JSON文字列
    if (typeof raw === 'string' && raw.startsWith('{')) {
        try {
            const parsed = JSON.parse(raw);
            if (parsed.selection !== undefined) return parsed;
        } catch (e) { /* fall through */ }
    }
    
    // 旧形式（プレーンテキスト）→ メモとして保持
    return { selection: '', memo: String(raw) };
}
```
