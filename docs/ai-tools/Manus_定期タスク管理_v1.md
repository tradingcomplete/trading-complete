## 🔧 Geminiとの連携

### 使い分け

| ツール | 役割 | 使うタイミング |
|--------|------|---------------|
| **Manus** | テキスト生成・リサーチ | 定期タスク（自動実行） |
| **Gemini - X画像クリエイティブ** | インフォグラフィック生成 | Manus出力を元に手動で画像生成 |
| **Gemini - 投稿最適化Gem** | 投稿文の最終調整 | 必要に応じて |
| **Gemini - Build in Public** | 開発進捗画像 | 週1回の開発報告時 |

### 画像生成フロー

```
1. Manusが市況テキストを生成
2. コンパナさんが確認
3. Gemini「X画像クリエイティブ・ディレクター」にテキストを入力
4. 16:9のインフォグラフィック画像を生成
5. Bufferでテキスト＋画像をセットで予約投稿
```
