### ステージ9：プロンプト調整・最適化ガイド

```
【CompanaTradingコンテンツ用プロンプト最適化フレームワーク】

1. A/Bテスト用バリエーション作成
各プロンプトに対して2-3のバリエーションを作成：
- トーンの違い（カジュアル vs プロフェッショナル）
- フックの違い（数字訴求 vs 感情訴求）
- CTA強度の違い（ソフト vs ダイレクト）

2. パフォーマンス測定指標
- エンゲージメント率（いいね、コメント、シェア）
- クリック率（プロフィール遷移、リンククリック）
- コンバージョン率（アプリダウンロード、会員登録）
- 滞在時間（ブログ、YouTube）

3. 最適化サイクル
週次で以下を実施：
- 数値分析（どのコンテンツが最も効果的か）
- ユーザーフィードバック分析（コメント内容の質的分析）
- プロンプト微調整（表現、構成、長さの調整）
- 新規テスト案の立案

4. プラットフォーム別最適化ポイント

【ブログ】
- SEO順位の変動をモニタリング
- 滞在時間が短い場合は、フック強化
- 離脱ポイントを特定し、構成を調整

【X/Twitter】
- インプレッション数とエンゲージメント率の相関分析
- リツイート率の高い投稿パターンを抽出
- フォロワー属性に合わせた投稿時間最適化

【YouTube】
- 視聴維持率グラフから離脱ポイントを特定
- サムネイルCTRを向上させるデザインテスト
- 終了画面からの次動画遷移率を測定

【TikTok】
- 完全視聴率を最重要指標に
- トレンド音源への素早い対応
- コメント欄でのやり取りを重視

5. 継続的改善のためのプロンプト更新履歴管理

バージョン管理テンプレート：
```
Version: 1.2
Date: 2024-03-15
Changes: 
- フック部分を質問形式に変更
- 数字を具体的に（500万円損失を追加）
- CTA位置を中間にも配置
Performance Impact:
- CTR: +15%
- 完読率: +8%
- CV率: +3%
```

6. コンテンツカレンダーとの連携
- 月間テーマに沿ったプロンプト事前準備
- 季節性やイベントに合わせた調整
- シリーズ物のストーリー整合性確保
```

### 記事テンプレートと執筆ガイドライン

#### 基本構成（3,000-5,000文字）

1. **導入部（300-500文字）**
   - 読者の悩みに共感
   - この記事で得られること
   - 読了時間の目安

2. **本文（2,400-4,000文字）**
   - ストーリー展開（時系列 or テーマ別）
   - 具体的なエピソード
   - データや数字の裏付け
   - 実践的なアドバイス

3. **結論部（300-500文字）**
   - 要点のまとめ
   - 読者への励まし
   - 次のアクション提案

#### トーン＆マナー
- です・ます調で統一
- 専門用語は必要最小限に
- 失敗談は具体的に、成功談は控えめに
- 読者を「あなた」と呼びかける
- 1文は40文字以内を目安に

#### SEO対策
- タイトルに主要キーワードを含める
- 見出し（H2、H3）を適切に使用
- 内部リンクで関連記事へ誘導
- 画像のalt属性を設定
- メタディスクリプションを最適化
