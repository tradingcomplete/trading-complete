## 🏛️ 引き継ぐ7原則（思想の核）

1. **Build in Public × 透明性**: 開発過程・失敗談・数字を堂々と公開する。信頼が最大の差別化資産
2. **フリーミアム**: 無料20件で価値を体験 → 自然に有料へ。Free→Pro転換率3%が現実的目標
3. **「記録してからトレード」=規律型オファー**: 「簡単」でなく「立ち止まる摩擦」を売る。競合の永続的差別化
4. **コミュニティファースト + 誠実さ**: 短期収益より長期信頼。アフィリエイトは透明開示・利益相反回避
5. **年払い推奨**: 月払いより年払い（実質2ヶ月無料）。キャッシュフロー改善＋解約率低下
6. **30代会社員に集中**: 「副業で人生を変えたい」「ルールが守れず損切りできない」層に刺さるメッセージング
7. **SEO + X + 個人開発の三本柱**: 広告単独は失敗事例多数。**SEO記事 → X毎日発信 → 補完として有料広告**の順

## 💰 アフィリエイト戦略

### 基本方針

**Phase 1（リリース時〜半年）**: CPA（口座開設単発報酬・5,000〜20,000円/件）で開始。
**Phase 2（半年〜1年）**: 安定したらIB（継続報酬型・1ロット200〜500円）に移行交渉。
**Phase 3（1年〜）**: IBが収益主軸へ。

### 候補ブローカー（リリース時）

DMM FX / GMOクリック証券 / 外為どっとコム / みんなのFX 等の主要日本ブローカー3〜5社。
※「FX」キーワードに敏感な決済プロバイダ審査リスクは検証済（v3.9 §5）。掲載は問題なし。

### 透明性と誠実さ

- すべてのページに「このリンクで開発を支援」と明記。手数料はユーザーに無関係を明言
- アフィリエイト報酬目的での粗悪ブローカー推奨は絶対NG
- 非アフィリエイトのブローカーも併記（ユーザー選択肢の提供）
- AI分析結果は「投資助言ではない参考情報」と必ず明示

### リスク管理

- 利益相反の印象を与えない（透明開示で対策）
- アフィリ単独で顧客獲得は困難（個人開発SaaS事例より）→ 補助収益と位置付け
- 法務確認: 金融サービス紹介は景品表示法・特商法準拠を継続確認

## ❌ あえてやらないこと（リソース集中のため）

| 施策 | 理由 |
|---|---|
| Meta（FB/IG）広告 | 30代男性へのリーチ精度がX < Google。個人開発SaaS成功事例少 |
| LINE広告 | FXトレーダーへのターゲティング精度不足 |
| B2B契約 | 初期リソースなし。ユーザー基盤確立後の検討 |
| ライフタイムディール（LTD） | 単発収益・長期価値毀損リスク |
| 複数インフルエンサー大規模契約 | テスティモニアル蓄積後の検討（半年後以降） |
| データベース販売 | 匿名化・利用規約の複雑化。1年後以降 |
| YouTube動画の本格量産 | 制作コスト過大。SEO+Xを優先（週1ペースで十分） |
| Indie Hackers英語コミュニティ深掘り | 英語版準備後 |

すべて「やる価値はあるが、優先度が現状で低い」もの。半年〜1年後に状況を見て再評価。
